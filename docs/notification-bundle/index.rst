Notification Bundle
===================

The *PerformNotificationBundle* provides ways to send notifications using various publishers.

* Send notification emails
* Show in-app notifications
* Abstract who to send to with the ``Recipient`` and ``RecipientProvider`` interfaces
* Configurable and translatable templates for each notification type
